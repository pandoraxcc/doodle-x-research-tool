The visual representation of local network activity

Allows you to track the packet travel from your local router to the destination host. Along with ip and hostnames, sends the request to ipapi.co service to resolve location data, including country, city, region, country area and organization name, as the service provider for the traced IP.

For the better visual and understanding, builds the graphs with chart.js library: pings, countries, organizations and adresses types.

Making network security analysis easy for everyone

With the help of open source tool Nmap, the tool brings you the scan of vulnerable/open ports, connected devices and other important security information of your local network.

The DoodleX tool is the visual wrapper of the most common and helpful commands widely used by cyber security specialists.

Technologies and tools

Nmap - Supports dozens of advanced techniques for mapping out networks filled with IP filters, firewalls, routers, and other obstacles. This includes many port scanning mechanisms (both TCP & UDP), OS detection, version detection, ping sweeps, and more. Current project uses python-nmap package which you can check here

Django and Chart.js - Django is the core stack, handling commands and performing the scans on behalf of a user, processing the requests and providing the outputs. Used for potential scalability of DoodleX tool and the new features. Chart.js - for easier visual representation of the data using jQuery and Javascript.
